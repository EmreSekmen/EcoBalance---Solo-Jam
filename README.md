# EcoBalance

EcoBalance is a small ecosystem simulation game developed in Unity.

## Features

- 🌱 Random Plant Spawner
- 🐭 Mouse AI
- 🦊 Fox AI
- 🍖 Hunger System
- ❤️ Reproduction System
- ⚡ Energy System
- 📊 Animal UI Counters
- 📦 ScriptableObject-based animal stats
- 🤖 NavMesh AI

## Planned Features

- Pause Menu
- Save / Load System
- Audio
- Settings
- UI Polish

## Technologies

- Unity
- C#
- NavMesh
- ScriptableObjects
